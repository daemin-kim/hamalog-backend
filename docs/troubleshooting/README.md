## 📝 빠른 시작

### 새 트러블슈팅 로그 생성

1. 파일 생성: `TSL-{번호 4자리}-{케밥케이스}.md`
   - 예: `TSL-0001-connection-pool-exhaustion.md`

2. 템플릿 복사: [DECISION-TROUBLESHOOTING-GUIDE.md](../internal/DECISION-TROUBLESHOOTING-GUIDE.md#31-트러블슈팅-로그-템플릿)

3. 위 목록 표에 새 문서 추가

### 상태 표시

| 상태 | 의미 |
|------|------|
| 🔴 Investigating | 조사 중 |
| 🟡 In Progress | 해결 진행 중 |
| 🟢 Resolved | 해결 완료 |
